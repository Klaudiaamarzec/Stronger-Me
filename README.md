# Personal-Trainer
A web application that makes it easier to find a personal trainer

The project aims to create an application designed to make it easier to find a personal trainer and to create a local community of athletes. It will make it easier for inexperienced people to start their adventure with the gym by working with the right personal trainer or arranging a joint training session. The system will make it easier for personal trainers to find clients, promote themselves, and manage their class schedules.
